 Order Management System (Sistema de Pedidos)
Um sistema web Full Stack para gerenciamento de pedidos e controle de estoque. O projeto permite cadastrar diferentes categorias de produtos (eletrônicos e perecíveis), aplicar regras de negócio específicas para cada um e gerenciar o carrinho de compras dos usuários.

Tecnologias Utilizadas
Backend (API REST):

Java 21

Spring Boot 3

Spring Data JPA / Hibernate

MySQL (Arquitetura com herança de tabelas - Joined)

Frontend (Interface Web):

React 19

Next.js 16

TypeScript
